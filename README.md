# 场地预约系统

An easy, simple and speedy reservation system.

## 开始使用

本指南将指导你使用本系统或自己构建一个快速简洁的场地预约系统。

### 前置要求(使用)

一台电脑以及畅通的网络。


### 前置要求(自行构建)

Jdk1.8+ 本项目基于Jdk11构建，之前版本也许会遇到一些问题。
